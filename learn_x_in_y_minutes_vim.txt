What is Vim?|<a href="http://www.vim.org/">Vim</a> (Vi IMproved) is a clone of the popular vi editor for Unix. It is a text editor designed for speed and increased productivity, and is ubiquitous in most unix-based systems. It has numerous keybindings for speedy navigation to specific points in the file, and for fast editing.
What is vimtutor?|vimtutor is a an excellent application that teaches you how to use Vim. It comes with the vim package during installation. You should be able to just run “vimtutor” on the command line to open this tutor. It will guide you through all the major features in vim.

How can vim be thought of?|"Vim can be thought of as a set of commands in a ‘Verb-Modifier-Noun’ format, where: <br>
Verb is your action <br>
Modifier is how you’re doing your action <br>
Noun is the object on which your action acts on <br>"
What are a few important examples of ‘Verbs’?|"d # Delete <br>
c # Change <br>
y # Yank (copy) <br>
v # Visually select"
What are a few important examples of ‘Modifiers’?|"i # Inside <br>
a # Around <br>
NUM # Number (NUM is any number) <br>
f # Searches for something and lands on it <br>
t # Searches for something and stops before it <br>
/ # Finds a string from cursor onwards <br>
? # Finds a string before cursor"
What are a few important examples of ‘Nouns’?|"w # Word <br>
s # Sentence <br>
p # Paragraph <br>
b # Block"

What are macros?|Macros are basically recordable actions. When you start recording a macro, it records every action and command you use, until you stop recording. On invoking a macro, it applies the exact same sequence of actions and commands again on the text selection.
